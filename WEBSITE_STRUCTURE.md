## 🎨 Color Palette Reference

### Primary Colors:
- **Charcoal**: #1a1a1a (Dark text, buttons)
- **Amber**: #c8956c (Accents, highlights)
- **Warm White**: #faf8f5 (Main background)

### Secondary Colors:
- **Cream**: #f0ebe3 (Light backgrounds)
- **Slate**: #6b7280 (Subtext)
- **Border**: rgba(200,190,175,0.3) (Dividers)

### States:
- **Error**: #e53935 (Form validation)
- **Success**: #2e7d32 (Form submission)
- **Hover**: Amber with shadow

## 📐 Responsive Breakpoints

### Desktop (1024px and above)
- 3-column service grids
- Side-by-side layouts
- Full navigation visible
- Multi-column footer

### Tablet (768px - 1023px)
- 2-column grids
- Balanced layouts
- Hamburger menu
- Optimized spacing

### Mobile (480px - 767px)
- Single column layouts
- Stacked forms
- Touch-optimized buttons
- Hamburger navigation

### Small Mobile (below 480px)
- Extra padding/margins
- Large touch targets
- Simplified layouts
- Optimized typography

## ✨ Animation & Effects Guide

### On Page Load:
- Fade-up animations on sections
- Staggered card reveals
- Counter animations (stats)

### On Scroll:
- Elements fade in as they enter view
- Parallax effect on hero image
- Cards lift on scroll reveal

### On Hover:
- Buttons change color/lift
- Cards lift and shadow deepens
- Links underline
- Images zoom slightly

### On Interaction:
- Form fields show amber border on focus
- Error fields turn red
- Hamburger menu animates
- Form submit shows loading state
- Success state after submission
